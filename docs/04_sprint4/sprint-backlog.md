## 設計メモ: 削除の誤操作防止(JS なし)

```
詳細ページ [削除] リンク(GET)
  → 確認ページ /todos/{todo}/delete 「本当に削除しますか?この操作は取り消せません」
    → [削除する] フォーム(DELETE /todos/{todo})→ 一覧へ+フラッシュ
    → [キャンセル] リンク → 詳細へ戻る
```

- 削除の実行は必ず DELETE メソッド(GET でデータを変えない原則)
- 確認ページ自体は「見るだけ」なので GET でよい
