# TODOアプリを作成するためのプロンプト生成

以下の内容でTODOアプリを作成するためのプロンプトを生成して。

## TODOアプリの作成

経験の浅い初級エンジニアがスクラム開発でTODOアプリを作成する。
必要な機能はCRUDで、技術スタックは下記リストを参照。

- Docker
- bitnami/laravel
- Plain HTML/CSS
- MySQL
- GitHub

スクラムは全４周とし、環境構築からスタートして必要機能をすべて実装することでゴールとする。
- product roadmap
- user story
- product backlog
- sprint backlog
- sprint review
- sprint retrospective

開発前に不明点を確認すること。またやりとりの要点は文書にまとめること。
各スプリント期間においても、計画・振り返り等、文書化を徹底すること。
経験の浅い初級エンジニアにとって教育的価値の高いものとなるよう意識すること。
コミットの粒度とメッセージの内容に気をつけること。
